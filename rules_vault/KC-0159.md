## Code Eco’s Headware

The goal here is obtaining Code Eco’s headware drive, which is located in the converted berth on the port side of the houseboat. The headware has extensive paydata on not only Okoro’s operations but also Ares’ extensive illegal arms production and distribution, as well as schematics for early versions of Evo’s anti-gravity tech. The paydata is worth quite a bit. Okoro’s data is worth 5,000 nuyen; Ares’ data is worth 10,000 nuyen; Evo’s data is worth 15,000 nuyen.

Most important to the investigation is the message from Code Eco to Okoro with a file attachment name “Kechibi Code.” The attachment is no longer present, but it confirms where Okoro got the code. The headware also has another file titled “KC Analysis” that indicates Code Eco’s findings that the Kechibi Code is some kind of super-analysis program intended to scan the global economic markets. He indicates he does not recognize the base coding and finds it far more complex than anything else he has ever seen before. He indicates in his filing system that the Kechibi Code was pulled from a cluster of downloaded data from an infobroker named Micawber in London.

The gamemaster can also use the headware to reveal that Code Eco’s death was not as random as everyone has been saying. A recording of his last moments can be found buried deep in the headware, and while the runners may not be able to identify the car that ran him down, they may see someone in the crowd, like an Area Boy leader or a prominent hawala, who could have been there to make sure Code Eco was properly struck.
