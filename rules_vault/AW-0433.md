## Random Miggon Mutation

When a character mutates, determine which attribute is affected randomly. If Magic or Resonance is rolled and the character does not have a Magic or Resonance attribute, no mutation occurs this time. If you roll an attribute for which you already have two mutations, no further mutation occurs at this time.
