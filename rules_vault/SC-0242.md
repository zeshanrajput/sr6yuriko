## Additional character creation recommendations

- **Attributes:** Charisma 4+, Willpower 4+

- **Dissonance aptitude:** Alt/Morphinae

- **Skills:** Con, Electronics (Software)

- **Knowledge skills:** Matrix Cryptology, Matrix Database Architecture, Software Development Research

- **Complex forms:** Editor, Pulse Storm, Resonance Veil, Static Bomb

- **Positive qualities:** Soothing Static, Sprite Affinity (Crack Sprites)

- **Negative qualities:** Dissonance Deletion, Dissonance Static, Dissonance Stigma



- **Life modules:** Activist, Data Liberator, Hacker, IT Support, Technical School, Technomancer, Event: You benefited from the influence of a mentor.
