## Urge Stopper

- **Vector:** Slot

- **Speed:** Immediate

- **Duration:** (Rating) hours

- **Power:** 7 + (Rating)

- **Effect:** Temporarily reduces the Compulsion negative quality by one level

This personafix helps alleviate compulsive behaviors. The BTL inserts memories of the user indulging in their compulsion and triggers a series of neurochemical responses to help delay the time where the user would need to repeat the behavior. These BTLs are custom-made for individual users and specific behaviors. This is generally a prescribed drug for obsessive-compulsive behavior patients. When using the p-fix, the user reduces the Compulsion negative quality (p. 129, _Firing Squad_ ) by 1 level. When the BTL duration ends, the user immediately needs to indulge in their chosen activity.
