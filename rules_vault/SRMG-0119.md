## **Can I install any cyberware using the capacity in different cyberlimbs?**

No. The type of base cyberware still needs to match the basic concepts of bodyware, headware, eyeware, etc. to fit in the item. You can’t put a select sound filter into your cybereyes, for instance. And any cyberware installed cannot be spread between multiple limbs’ capacity.
