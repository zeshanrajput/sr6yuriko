## Framework Hosts

Framework hosts are more flexible with the attributes. One attribute can be set as low as –1 below the rating, and the +6 (or +7) points can be allocated in any pattern but limited to +4 on a single attribute. For example, a possible set-up could be (Host Rating –1), (Host Rating), (Host Rating +3), (Host Rating +4).
