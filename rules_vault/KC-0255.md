## (K405)

One of the more unusual groups I’ve run into on my search for the code is K405, and I find them especially dangerous because it seems their onlymotivation is to use the code to cause … well, chaos. K405 is an Apophenian tribe whose members have dedicated themselves to an entity they call the Algorithm, which seems to be a dissonant Paragon. What I know about the Algorithm is that it knows nothing and cares nothing about metahuman lives and existence. It prioritizes the gathering and use of information only for its own chaotic ends with no regard to anyone its influence touches; it wants you to buy things, to engage in repetitive behaviors on the Matrix solely because that’s what it does, and it doesn’t give a drek about how it’s going to affect you. Produce, engage, consume, repeat; the Algorithm wants to trap you in this incessant loop.

If K405 didn’t create the Kechibi Code, and if they’re not already worshipping it like some kind of fragged-up Matrix god, the kind of things they might do with the Code would unleash catastrophe on the Matrix and the rest of the Sixth World.

A technomancer friend of mine was telling me a story about this the other day. My friend was out making some cash when they ran across some of K405’s efforts. K405 had created a honey-pot host, spreading rumors that they had a prototype of the code and were working to make it functional. Sure enough, Renraku took the bait. My friend isn’t exactly sure what happened next, but when my friend ran into a Renraku host as a part of a run,

Renraku were trying to disassemble and examine some code that they had captured from K405. I don’t know exactly what a “whirling maelstrom of dissonance” looks like exactly, but my friend said that’s what they saw, right before they decided to move their meat to the Ozarks for a while to try to live off the grid. I hope they’re doing okay.

In any case, I don’t know if K405 really do have any of the actual code, but it looks like they are using it as bait, and disguising weird Dissonance bombs as parts of it. So if someone offers to send you the code over the Matrix, maybe get a technomancer to give it a once over before clicking that attachment.

Being able to mess up technomancers with Dissonance weapons, though? That kind of tech would be priceless.
