## Vector Thrust Wings

If you want to provide a critter that can’t fly with the ability to fly, attaching vector thrust wings to an exoframe is one way to accomplish it. These wings are designed to allow a critter to glide and are also equipped with jet turbines. A specialized drone pilot system handles the details—the critter just needs to decide where to go.

**Game Information:** A critter must be equipped with a critterjack and receive special training in order to use this accessory. These wings allow a critter to glide, just as with the glider wings accessory, but the equipped jets also provide them with the ability to fly with a Handling 3, Acceleration 25, Speed Interval 50, and Top Speed 250. The vector thrust wings’ rating defines the Pilot rating of the drone pilot system and is hardcoded with a maneuvering autosoft with the same rating. For Piloting tests, use Pilot x 2 for the dice pool—no Edge may be used to influence this test, even if the critter has a heuristic processor. When landing, the critter must make an Athletics + Agility (2) test or fall prone. This accessory can also be used for shortrange jumps, in which case they function exactly as the jump jets accessory. Because these wings are too bulky to fold up very well, when not in use the critter gains the Hobbled status (p. 52, _SR6_ ). Flying requires a Minor Action from the critter each round, which counts as its movement action.

The included battery array provides enough power to fly for up to one minute before needing a recharge. Additional battery arrays take up 1 capacity and provide one additional minute of flight time each.
