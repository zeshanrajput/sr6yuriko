## **Can I write my own code and software (p. 55)?**

Yes, but note that this software cannot be sold. For the purposes of _Shadowrun Missions_ , programs you’ve written cannot be given or sold to other runners. For any software, the maximum rating is the same as purchasing the software. For example, autosofts cannot exceed rating 9. This is available during character creation.
