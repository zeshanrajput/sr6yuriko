## Inspire

## \[Influence (Leadership), Major Anytime\]: 3 Edge

This action is used to inspire an ally using an Influence (Leadership) + Charisma test. For everyhit, the target gains a bonus die on their next skill check. This counts as a Teamwork test, so the bonus dice are limited by the target’s ranks in the skill they’re going to use. If there are multiple Major Actions and enough Edge available, player characters can combine this with the Coordinated Effort Edge action. They cannot use this action to provide bonus dice when a target is the leader of a Teamwork test.
