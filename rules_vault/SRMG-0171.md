## **What can Full Matrix Defense be used on?**

This can only be used on Matrix defense tests vs. cybercombat and Resonance Spike. For a list of Matrix actions that it can be used against, see **What specialization and expertise apply to my Matrix Actions?** above for the specific actions associated with cybercombat.
