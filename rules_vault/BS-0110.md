## Matrix Visualizer

There are few tools better than a Matrix visualizer for constructing AROs and sculpting VR. This augmentation permits the user to “print” mental images as virtual Matrix iconography. A creation requires the user’s full concentration for a period ranging from a single combat round for a basic geometric design, to hours for an immersive landscape or true work of art. When attempting to render details from memory, the user rolls a Memory Test (p. 67, _SR6_ ) against an appropriate threshold (see **Threshold Guidelines,** p. 36, _SR6_ ).

**Wireless bonus:** A Matrix visualizer can sync with a wireless printer to create physical 2D or 3D objects.
