## **How does Software Optimization (6WC, p. 135) work?**

You must designate one program that Software Optimization gives its bonus Edge for using. “Use” of this program means a test that gets modified by the software. You must designate a single basic cyberprogram or a hacking cyberprogram to benefit from Software Optimization. For example: the armor hacking cyberprogram (SR6, p. 184) grants +2DR constantly, but Software Optimization only grants Edge when your Matrix defense rating is invoked for a test.

This does not work for commlink apps, codemods, skillsoft, autosofts, etc. as these are not cyberprograms.
