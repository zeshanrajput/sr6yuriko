## Retractable Climbing Claws

These claws are essentially hand razors that curl inward when extended. A single set grants Edge on Climbing tests (p. 93, _SR6_ ). If both hands are enhanced, using the pair together improves this bonus by reducing climbing thresholds by 1. An additional pair can also be installed in the feet, resulting in a total threshold reduction of 2 (to a minimum of 1). It takes a Minor Action to deploy or retract a set of claws. Although the hooks can potentially cause gruesome flesh trauma when used offensively, they are suboptimal close combat weapons. They inflict DV 2P with AR 4/—/—/—/—.

**Wireless bonus** : Gain a bonus Minor Action after spending one to extend or retract a set of claws.
