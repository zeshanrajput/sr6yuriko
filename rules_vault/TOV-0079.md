## Using Gris-Gris

Gris-gris require a Minor Anytime Action to activate. Users can activate multiple charms with a single Minor Action. The grisgris only stays active until the next test that the user makes. If the next test made is not appropriate to a specifi c activated gris-gris, that gris-gris fully deactivates without expending a service. Any time the gris-gris aids in a test or uses its power, a service is consumed—unless otherwise stated with the specifi c types of charms.
