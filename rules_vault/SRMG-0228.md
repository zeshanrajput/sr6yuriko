## **Are there any** _**Seoul Survivor**_ **gear errata for** _**Missions**_ **?**

Type 81: Availability is 2(I).

Bujeok: Only one of these can be used at a time as the use of multiple will cause them to interfere with each other. It takes a Minor action to affix a new one to your person.
