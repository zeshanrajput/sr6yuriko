## Matrix Security Operatives

## Security Spider (Professional Rating 4)

These are the run-of-the-mill rent-a-cops of the Matrix security world. Whether they are a contractor on hire monitoring the host from a remote location or a direct employee that is on site, you will find these guys in any host an owner cares enough about to spend money on non-automated Matrix security.
