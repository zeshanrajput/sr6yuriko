## **New Matrix Protocols:** See New Matrix.

- **Noosphere:** An energy field generated by information transmission and storage. This is based on the universal information theory.

- **Nous-Particle Field Mechanics:** The central component of universal information theory. This refers to how data and information are connected through hosts, the mind, and the Resonance Realms.

- **LAN:** Local area network. Refers to the geographic area where devices can be slaved to a small Framework host (Scale 1 or 2).

**PAN:** Personal area network. This is all the devices that are linked with your main device, usually either a commlink or cyberjack.

**Paradigm:** This is the term for the Foundation where the environment including all Foundation avatars and objects are forced to match some look and feel. This is similar to a host’s sculpting, except way more invasive.

**Quad, Cube, and Wall:** These are pre-networked packages of 4, 16, or 64 optical stack mainframes.

**Resonance Realms:** The generally unknown structures underpinning the Deep Foundation. Technomancers know about it. Go ask them.

**Scaffolding:** The system that runs the sculpting of a host. This is controlled by the scaffolding control node in the Foundation.

**Sculpting:** The programming that defines the user interface of a host. Like the paradigm, only it doesn’t force you to wear some silly outfit or suddenly transform into a car.

**Shell:** The host envelope before any sculpting is performed. It’s a host, but just barely.

**Transfer Circuit:** The code used by CC licensed corporations to create Foundation host shells. This somehow magically grows a host out of the Foundation. I am sure de la Mar knows more, but so far she’s being tight-lipped about it publicly.

- **Universal Information Theory:** Also known as UnITy, it is an attempt at a unified theory explaining Foundations, technomancer abilities, the Resonance Realms, and an assortment of other Matrix phenomena. Much of this research centers around the Noosphere.

- **Variance:** A term used for the difference between what a paradigm expects and the runners’ actions. This is bad because when the variance gets high, then the entire host Foundation is going to be out to get you.

- **WAN:** Wide area network. Used to describe the geographic area where devices can be slaved to a distributed Framework host (scale 3 to scale 6) and is basically a network of overlapping LANs.

- **Optical Stack Mainframe:** The standard high-power computing device for generating hosts.
