## Create Sha/Sheng Event

With True Earth, the initiate can also adjust the harmonics of the mana around the area, creating a mana flow or mana ebb. Using the same cleansing thresholds for a mana ebb, the initiate can create a mana ebb or mana flow of the listed Nature (p. 168, _Street Wyrd_ ) for a number of hours equal to the magician’s initiate grade; the size of the sha/sheng event is (Magic rating + Initiate Grade x 10 meters), and the type of the event (tradition specific or general) is up to the initiate.

This ability can be added to a metamagic cleansing focus created with True Earth.
