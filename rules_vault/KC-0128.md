## Behind the Scenes

They manage the resources gained through Okoro and Obukanwe’s use of the code, but Obukanwe has hidden an actual copy of the code offline, as she is afraid to use it again. She is considering selling it but doesn’t want the sale tracked back to her, especially with her growing paranoia. She’s met more and more shady people in connection with Okoro’s efforts and is currently grooming three of them to see if they’d be reliable agents to sell the code and prevent it from being traced back to her. Of course, like all the other offline copies, her version of the code is decaying, and it now has little actual worth, which will be a rude awakening for her should she finally sell it.
