## **Can I build my own host (p. 50-52)?**

Yes. Just remember that the host can't be used to hack things on the Matrix. You should be aware that the bigger the host, the more space and power it needs if it is a framework host. When the owner lacks the



lifestyle to maintain the host, the host is offline until the lifestyle required is restored.

| **Rating** | **Lifestyle** |
| ---------- | ------------- |
| 1-3        | Low           |
| 4-6        | Middle        |
| 7-9        | High          |
| 10-12      | Luxury        |
