## Core

The core represents a tangled cluster of interconnected processor and data storage chips that do all the hard work and tie all of a cyberdeck’s components together. This component provides the cyberdeck’s Device Rating. The rating of an individual Attack or Sleaze module cannot be higher than the deck’s Device Rating x 2.
