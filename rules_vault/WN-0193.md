## Take A Peek Behind The Curtain

The first job is a simple smash-and-grab. The Kente Royal Gallery is hosting an augmented reality event showcasing AR skins, which are commonly used to display a virtual set of clothing the user is wearing. During that event, the Gallery will be premiering an interactive piece of artwork called the “shifter” device that has the ability to redraw an individual’s AR skin using an algorithm based on their semiotics and available data profile. Just point and shoot in the real world and the AR skin changes to a version of that individual that most resembles what they present themselves as based on available data. The assignment is: Steal this device and return it to her.
