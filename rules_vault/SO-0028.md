## Game Information

Every shadowrun has some part of it dedicated to planning. Whether a team actually sticks to that plan is another issue. Yet, how does one know when they’re prepared enough and it’s time to execute? Sometimes planning a single run can span multiple sessions. If this method works, then keep doing it. But for those that want a more streamlined method there is the Heist Pool.
