## Dedicated Program Slot

## Core Slots: 0.5 each

A dedicated program slot is designed to store and run a specific cyberprogram, which cant be altered without changing out the components. A dedicated program slot runs more efficiently and takes up less space, but you can’t use it to run any cyberprograms other than the one it’s designed for. These are most often used for Baby Monitor or Agent software.
