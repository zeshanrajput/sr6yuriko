## Game Information

To function properly, this app requires the user to have an image link. With a Major Action, it allows the user to conduct a Matrix Search action in order to identify anyone who’s broadcasting their SIN, other types of ID, or credentials. If the user is also running the facial scanner app, with a major action they can attempt a Matrix Search action to identify individuals who aren’t broadcasting their SIN or credentials. But only those individuals with a SIN on file will return a result. Users of this app will gain a point of Edge on their first social skill test with an individual they’ve identified.
