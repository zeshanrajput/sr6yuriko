## New Sprite Commands

Technomancers tend to be creative, so it’s no surprise that new uses for sprites are discovered with time. Each of these new commands is specific to one or more types of sprite, which means they can only be used with a sprite of the matching type. These commands only work with registered sprites—newly compiled sprites just don’t seem to have the juice.

All of these commands are available to modular sprites. None of them are available to primal sprites.
