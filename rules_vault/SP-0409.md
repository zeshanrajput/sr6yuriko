## **Game Effect: Cyberlimb Plating**

**The GMT modifcation can be applied to any cyberlimb, cybertorso, or cyberskull. This takes up one capacity per limb for obvious cyberlimbs, and two capacity per limb for synthetic cyberlimbs. When incorporated into a cyberlimb’s original design by purchasing the mod at the same time the cyberlimb is purchased, the capacity cost is reduced by 1 (meaning that there is no capacity cost for obvious cyberlimbs) and the nuyen cost for the mod is ffty percent less.**

**For each cyberlimb with the GMT modifcation, the user gains +1 to DR and dice pools for defense tests against mana spells and critter powers that are resisted by Willpower, Intuition, Logic, or Charisma. These bonuses are cumulative with other GMT cyberware.**

**A cyberskull with the GMT mod provides additional protection against magic that targets the mind. Add +1 to the threshold when targeting the user with the following spells and critter powers:**

**Analyze Truth, Mindlink, Mind Probe, Control Thoughts, Compulsion, Confusion, and Infuence. This effect is not cumulative with GMT bone lacing.**

**A cybertorso with the GMT mod provides additional protection against magic that targets emotions. Add +1 to the threshold when targeting the user with the Fear critter power.**
