## Mirrored Persona

**==> picture [152 x 20] intentionally omitted \<==**

**----- Start of picture text -----**<br>
FADE VALUE DURATION<br>3 S<br>**----- End of picture text -----**<br>

You can create a proxy persona that looks and acts identical to you in the Matrix. When launching this form, roll Electronics + Resonance; the number of hits serves as the rating of the duplicate persona you create. When targeting the Technomancer with a Matrix Action, opponents must succeed on a Matrix Perception test with a threshold of that rating.

If they fail, opponents target the proxy with their Matrix Action and then Mirrored Persona ends.
