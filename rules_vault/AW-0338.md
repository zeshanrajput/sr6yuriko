## Airship Chases

For airship chases, use the chase rules in _Double Clutch_ (p. 173). If the airship is not under the control of the PCs, or if you’d prefer to simplify the airship-to-airship combat crunch, make a Piloting + Reaction test for each airship. Whichever airship wins the test gets to set the range between the two vessels. If the range is already Extreme, the winner of the next test may choose to escape.
