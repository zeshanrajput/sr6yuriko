## **Linked Attribute:** Agility

Some weapons are not like any other, and you need particular training to use them. This skill is for those weapons. You must select a specialization to use this skill, and you can only use this skill with weapons for which you have a specialization. Expertises cannot be acquired for this skill. You can buy ranks in this skill, and those ranks can be used with all specializations you have. This skill is exempt from the rule that you can only have one specialization per skill. This skill functions the same as other combat skills, meaning you roll Exotic Weapons + Agility vs. Reaction + Intuition; see the **Combat** chapter for more details.
