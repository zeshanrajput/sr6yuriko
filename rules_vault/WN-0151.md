## **Security Host**

**Theme:** A simple room of half-circle stations with monitors and controls on large panels. The number of stations depends on the number of security specialists on duty. During the day (06:00-18:00), there are two. Overnight (18:0006:00), there is only one.

**RATING A S D F** 6 9 6 7 8

**IC:** Patrol, Blaster, Acid, Binder, Tar Baby, Sparky **Notes:** This is a hub host. Data may be collected and worked on in the office host, but it still passes through the security host before getting to the data storage host. The security host monitors access to the Valentine offices and access to all data on the data storage host.
