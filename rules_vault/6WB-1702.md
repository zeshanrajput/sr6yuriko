## Off the Grid

You are well-versed in using the many dead zones, static noises, outdated protocols, and of course the ancient, cable-bound Matrix of Berlin’s Eastern anarcho zones to mask your decking exploits. This is not a skill anyone can learn – you have to be born to this, “feeling” the right current to take, “smelling” the failing firewall of a passing pizza delivery drone to hop onto the next wifi connection, and “sensing” the tourist that leans against a public chip telephone, inviting you to use his commlink for a quick signal boost.

- Districts: Gropiusstadt, Köpenick, Kreuzhain, Lichtenberg, Marzahn, Pankow

- Cost: 8 Karma

- Game Effect: When hacking while physically located in one of the listed districts, reduce OS by 1 for every illegal action performed, and gain one bonus point of Edge both when using the Hide Matrix Action and when defending against Track IC. This bonus Edge is lost if not spent on that test.
