## Onotari Interceptor

Saeder-Krupp developed this multipurpose missile launcher for military use. It has two separate barrels that can load two missiles at once, which don’t need to be the same type. Both barrels cannot be fired simultaneously, though this safety feature has been known to be removed, much to the dismay of the target—and the firer, as it turns out. It features an internal smartgun system that controls the fire-safety system. Firing both barrels launches both weapons. Split the attack dice pool and resolve each projectile independently. The firer must resist 6P Fire damage when launching both tubes.

**Wireless bonus:** The safety system cannot be disengaged, but the firer gains the smartlink wireless bonus.

using the Hit the Dirt Action to reduce damage, as the Prone status offers no benefit against the airburst.

**Wireless bonus:** An airburst link requires wireless functionality to function at all. Both the grenades and the launcher must have wireless mode turned on for the system to function.
