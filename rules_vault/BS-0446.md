## Infiltrator

- **Vector:** Slot

- **Speed:** 1 combat round

- **Duration:** (Rating) hours

- **Power:** 9 + (Rating)

- **Effect:** +2 Agility, +1 Strength, –2 Charisma, Athletics 2, Engineering 1(Locksmith +2)

The original version of the infiltrator personafix was loaded up with trid memories of famous spies, but it turned out that only terrible spies became famous. More modern versions of this add a filtering layer on sensory inputs, highlighting only those who are relevant to the task at hand. Unfortunately, that leaves little room in the program for social cues. The user of this p-fix needs to make a Composure (3) test for them to resist the urge to steal any unattended items, or for them to be able to leave a heist in progress. This p-fix reduces the Edge cost of the Called Shot: Incapacitate (p. 93, _Firing Squad_ ) by 2. When the BTL wears off, the user is paranoid for the next (rating) hours and cannot use Edge on any Social tests.
