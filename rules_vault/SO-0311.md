## Game Information

Provided an olfactory sensor and the expenditure of a Minor Action, the user can gain a point of Edge on a Judge Intentions test. This sensor can be defeated by specific perfumes and cologne designed to overwhelm the software. This software can also tell the user what brand of tailored pheromones a person is using.
