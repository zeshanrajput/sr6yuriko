## Simsense Interface Overdrive

This augmentation adds parallel simsense processors to the brain, amplifying reflexes and associated functions. Each rating of this cyberware adds to the user’s Intuition. Since the augmentation must accept simsense input, this bonus only applies while in a VR Matrix User mode.
