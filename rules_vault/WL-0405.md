## **Dogpile**

## **(Minor Action: Command Critter)**

**In order to use this Edge action, all the affected critters must know an attack command. With one Minor Action, you command your critters to attack as a group, concentrating their attack against one target at a time. If all of your critters know the Pack Attack command, this Edge Action is not needed.**

- **Cost: 1 Edge per critter (Critters with the Pack Attack command don’t add to the Edge cost)**
