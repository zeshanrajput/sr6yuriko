## **GMC Elk**

**A fairly s tandard snowmobile, the Elk’s only true modifcations are the lightbar/siren and searchlight for rescue operations. Used in cold climes exclusively, the Elk is a gas-burning machine with a multifuel modifcation that allows it to run on virtually anything—alcohol, animal fat, even wood if you’re desperate enough. In addition to ensuring that it can be trusted to run no matter how bad the weather, drivers soon learned that they could add extra chemicals to the fuel mixture to send out colored smoke. This allows rescue teams or rangers to broadcast their location so lost hikers can more easily be rescued.**
