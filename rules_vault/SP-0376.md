## **Memory Crystal: Game Rules**

**Memory crystals have ratings from 1 to 6, with the quality of the crystal directly relating to the owner’s status within Dis society. The Firewall and Data Processing attributes of memory crystals are equal to cyberjacks with the same rating. While memory crystals can be hacked in a limited way by technomancers, only a native Disian or one of their servant races used for information processing (such as the Refugee in The Third Parallel ) can make full use of a memory crystal.**
