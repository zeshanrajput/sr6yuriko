## Hacking

- **Armor:** +2 to Defense Rating.

- **Biofeedback:** Causes Stun (if target is in cold-sim) or Physical (if target is in hot-sim) damage with Matrix attack. This program is linked to the Attack attribute.

- **Biofeedback Filter:** Allow Device Rating or Body roll to soak Matrix damage.

- **Blackout:** Like Biofeedback, but Stun damage only. This program is linked to the Attack attribute.

- **Decryption:** +2 dice on Crack File action.

- **Defuse:** Allow Device Rating or Body roll to soak damage from Data Bomb.

- **Exploit:** Reduce Defense Rating of hacking target by 2.

- **Fork:** Hit two targets with a single Matrix action without splitting dice pools.

- **Lockdown:** Cause link-lock when you do Matrix damage.

- **Overclock:** Add two dice to a Matrix action.
