## **mindspeech**

**This ability allows a user to communicate with another individual that is within line of sight. The ability can cross a single medium, meaning someone who is using astral perception can use the ability on a subject that is upon the astral plane. An individual using a simsense VR connection (cold or hot), can use the ability to communicate with another persona that it can perceive within the Matrix environment. Should the agent be a monad (p. 76, “Collapsing Now,” or p. 149, “Whisper Nets”), they can use the ability as any other individual might with the same restrictions. AIs and EIs can use the ability with any being that has active Matrix access. Matrix agents cannot use the ability upon a being that is not in the Matrix, and it doesn’t work through video or cameras. The ability supersedes the necessity of a shared language.**
