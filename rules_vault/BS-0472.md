## Comprehension Test

When a critter is issued a complex order, or if the situation changes while the critter is attempting to carry out a command, the critter must make a Comprehension test using its Logic + Intuition. The threshold for the Comprehension test is usually 1, but if a complex order involves multiple commands or the GM determines that the situation is particularly confusing for the critter, the threshold increases to 2.
