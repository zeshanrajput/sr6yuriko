## Neuro-Retention Amplification

This bioware provides the user with superior short-term memory. It’s a stripped-down version of the mnemonic enhancer with a fancy title, truth be told. Less effective, but handy. The user gains the Photographic Memory quality (p. 73, _SR6_ ).
