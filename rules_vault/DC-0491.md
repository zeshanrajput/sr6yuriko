## Improved Cargo Space

Much like Improved Seating, this gives the vehicle more cargo space by adding compartments. You still have the issue that there is only so much you can add without increasing the vehicle’s Size Class. Each level adds 2 CF. Maximum increase is double the starting CF or 3 levels whichever is less. Cost: 2 per level.
