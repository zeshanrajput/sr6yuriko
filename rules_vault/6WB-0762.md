## devices, AR, and VR

The Attributes you’ll use for Control Device Tests may change based on whether you are in AR or VR while using it. If you are using a device via Augmented Reality, and the device requires a Physical attribute, use your normal Physical attribute for this test. If you are using this device via Virtual Reality, and the device requires a Physical attribute, substitute the appropriate Mental attribute for this test. (Body = Willpower, Agility = Logic, Reaction = Intuition, Strength = Charisma).
