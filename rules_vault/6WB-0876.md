## Sprite Powers

## Camouflage

The sprite can conceal a file within another file in such a way as to make it invisible to Matrix searches. Concealed files can only be found with a Matrix Perception test that is specifically looking for the hidden file; even the sprite must make this test in order to find and extract it.
