## Scaling

Unlike a lot of cons, there isn’t a lot of escalation available. It exists in a popular niche for runners, but it’s still a niche. The most obvious scale down is meat-and-potatoes infiltration, like taking the guise of the vending machine delivery or, with more prep, a janitor. The quick and dirty version if the place has a loading dock is to wait for a delivery. Blending in isn’t usually too hard, and a box on the shoulder can block the face from the correct angle. It’s hardly a long con, but if you just need to get in the door, it’ll do.

- It’s also good to remember that the Fire Drill isn’t just a way to get in, but a viable way to get out. If security isn’t line-of-sight in pursuit, you might be able to blend into a crowd this way.
