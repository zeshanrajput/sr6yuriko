## Diseases

Diseases in _Shadowrun_ operate essentially like toxins and use the same base set of rules (p. 121, _SR6_ ). However, there are a couple of extra pieces to be aware of with them. This discussion contains disease rules that are relevant to Infection with HMHVV and a more detailed set of rules for diseases in general.
