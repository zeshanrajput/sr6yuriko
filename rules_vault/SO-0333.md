## Game Information

A pixel tag recovery kit is a small handheld device with replaceable reservoirs for storing pixel tags. It vacuums up dust and other small particles, filters them for pixel tags, and stores the pixel tags in a storage reservoir, usually ejecting the waste via a vent. Though some models include a second reservoir for capturing the waste. Recovering one square centimeter of pixel tags with this device takes a Minor Action. Each storage reservoir is capable of storing up to 100,000-pixel tags.
