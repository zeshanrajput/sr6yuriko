## Detonator cap

This detonator is inserted into a mass of explosives and set off either by a programmable timer or radio signal. Setting the timer takes a Major Action.

**Wireless bonus:** You can set the detonator’s timer with a Minor Action. You gain a Minor Action Command DNI-linked device Action to detonate the explosive.
