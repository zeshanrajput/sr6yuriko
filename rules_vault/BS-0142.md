## Gecko Tips

Gecko grip technology has long been commercially available, and gecko tips apply it to cyberware. Gecko tips function as gecko tape gloves (p. 280, _SR6_ ) for the affected hand. If both hands are augmented with gecko tips, the user benefits from assisted climbing (p. 93, _SR6_ ). It takes a Minor Action to activate or deactivate a limb’s gecko tips. All bonuses from activated gecko tips, including cyberware overdrive, only apply while climbing or holding dry, non-greased surfaces.

**Cyberware overdrive:** If the user has gecko tips on two hands and two feet and they are all wirelessly enabled while using cyberware overdrive, the user gains a non-magical version of the Wall Running adept power (p. 158, _SR6_ ). Convert one die on the Sprinting test to a wild die.

**Wireless bonus:** Gain a Minor Action after spending one to activate or deactivate a limb’s gecko tips.
