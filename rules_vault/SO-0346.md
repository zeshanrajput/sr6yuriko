## Interrogation and AntiInterrogation Gear

## Wakabayashi Technologies Emotional Analyzer 7000

This is one of the key devices used by security personnel. The package includes glasses with an augmented reality display that contains cameras, olfactory, and audio sensors. There is a briefcase-sized computational unit that includes the main processing capabilities advertised as having all the relevant smarts and the intelligence necessary to read any perp in real time.

Using this hardware and software solution will provide up to 4 Edge on a Judge Intentions test. The user will gain 1 Edge for each of the following sensors that have not been blocked: Cameras, Microphones, Olfactory, and Atmospheric.
