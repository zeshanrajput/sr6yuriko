## PAN Emulation

At the basic rating 1 level, this codemod creates a simple commlink-style personal area network for the AI, allowing for a number of devices equal to the AI’s Data Processing attribute to be linked for remote operation. Any number of devices can also be connected to a PAN for protection. Rating 2 functions more like the PAN generated by an RCC, and the number of linked devices increases to 3 x the AI’s Data Processing attribute. At rating 3, this codemod operates like a true command
