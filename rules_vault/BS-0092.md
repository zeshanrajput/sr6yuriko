## Rolling Audio Recorder

Cyberears can be equipped with a function that digitally records a minute’s worth of audio in a self-overwriting perpetual loop. By spending a Minor Action, the user can permanently save the last minute’s worth of audio heard by the cyberears.

**Wireless bonus:** If the user also has a wireless-enabled rolling video recorder, both augmentations can be commanded to save the last minute’s worth of data with the same Minor Action.
