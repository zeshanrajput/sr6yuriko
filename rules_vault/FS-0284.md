## Hip Pad Bracing System

| **AVAIL** | **COST** |
| --------- | -------- |
| 2         | 180¥     |

**The accessory alters the AR as follows:** +2/+1/–1/–2/–4, and it provides a +2 bonus to Initiative Score in the next combat round after it was used. The Aim Action cannot be used while firing with this system. If a weapon has no Attack Rating in a given range category before this modification is applied, that category is not affected. **Slot:** Stock
