## Turret Ammunition Storage Module

In order to provide additional ammunition capacity for turret-mounted weapons, this storage module contains extra bullets and an ammo feed system so you aren’t limited to the original weapon’s ammo capacity.

**Game Information:** An ammunition storage module provides additional ammo capacity based on the type of weapon it is designed for. Each point of exoframe capacity can store up to 50 darts (taser or dart pistol rounds); 250 pistol, machine pistol, or submachine gun rounds; 100 rifle or machine gun rounds; 50 shotgun shells; 10 assault cannon rounds or launcher grenades; or 1 missile/rocket. The module can only store ammo for one type of weapon. Multiple modules can be attached to allow greater flexibility in types of ammunition or to provide extra ammo for multiple turret mounts. Switching from one ammunition storage module to another takes a Minor Action.
