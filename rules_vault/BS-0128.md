## Skill Hardwires (Rating 1-6)

Skill hardwires are a cheaper alternative to standard skillwires. This dedicated system is implanted with a hard-coded skillsoft that is already included in the price. The rating of the artificial skill equals the skill hardwire rating. The software cannot be changed, but multiple skill hardwires can be implanted without interfering with each other or with skillwires. Because the system includes the necessary processors, skill hardwires do not require a skilljack. As with standard skillwires, the user cannot gain or spend Edge on tests involving this artificial skill.

**Cyberware overdrive** : Skill hardwire rating increases by +2, ignoring the maximum rating. One of these gained skill dice is converted to a wild die.

**Wireless bonus** : The user can gain or spend Edge while using skill hardwires.
