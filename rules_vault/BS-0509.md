## Cargo Module

External cargo cases allow the critter to carry supplies, tools, weapons, or anything else that might be needed. Unless the critter has arms (or a mechanical arm module), it cannot access the cargo on its own.

**Game Information:** The cargo module can store items in it, with its capacity determined by its rating. Each point of capacity provides enough space for an item that is roughly the size of a pistol. Items stored within the cargo module are protected from harm and are not damaged unless the cargo module is intentionally targeted using a called shot or caught in an explosion (or indirect area-effect Combat spell). The Matrix shielding upgrade blocks all wireless signals to and from the inside of the cargo module when it is closed.
