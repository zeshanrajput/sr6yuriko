## Transforming Garment

This garment can be designed to work in conjunction with mobile fabric components and features electrochromic capabilities so that it can transform into several distinctly different outfits upon command. Ideal for making just the right impression while going from one meeting to another, but even more useful when a quick disguise is needed.
