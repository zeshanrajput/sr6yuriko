## Building Your Pool

Building your Heist Pool is accomplished by doing normal legwork. This could involve talking to a contact who happens to know the catering details for an event your team is going to infiltrate. Or maybe a Matrix specialist is working to get the schematics for the electrical system so it can be sabotaged. Points in this pool represent exceptional results for an avenue of investigation.

Typically, legwork is done by calling up contacts, searching the Matrix, staking out a site, breaking into a host, or any other number of activities with the explicit purpose of gathering info or setting up contingency plans. When using contacts (a Connection + Connection test, p. 50, _SR6_ ), knowledge skills (a Memory test or Logic + Intuition test, p. 67, _SR6_ ), or a Matrix search (Electronics + Intuition test, p. 183, _SR6_ ) for the initial exploration, if the result is four or more net hits on the test (for the case of Matrix search, this is measured only on the first interval of the extended test), then you’ve earned a Heist Point.

In addition, gamemasters should provide a Heist Point for highly relevant discoveries the team can use as part of the run. This applies to specific legwork to dig up data on hosts, visiting the location, or other preparatory activities.
