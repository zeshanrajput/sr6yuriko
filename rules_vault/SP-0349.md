## **Reconnaissance**

**This next category is another group utilizing animal forms. Disians appear to be able to acquire vast quantities of information about a target in a short amount of time by deploying large groups of these recon chimera. The memories of these constructs can be downloaded into a memory crystal where it can be copied, processed, and shared.**

**Skills: Athletics 5 (Flight +2), Close Combat 4, Perception 6, Outdoors 4, Stealth 5 (+2 Visual)**

**Qualities: Photographic Memory**

**Powers: Animal Control (ordinary birds), Armor 2, Enhanced Sense (Audio Enhancement, Low-Light Vision, Select Sound Filter 3, Spatial Recognizer, Thermographic Vision, Vision Enhancement), Innate Spell (Mindlink), Mindspeech, Natural Weapon, Sapience, Venom, Weakness (Purple Blood) Optional Power: Search Attacks:**


**always gets two hits. Due to the Photographic Memory quality, a Disian can remove data from the chimera through the Mindlink and transfer it to a memory crystal for storage.**
