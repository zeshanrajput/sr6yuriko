## Keypad Sequencing Fingers (Rating 1-6)

This implant replaces all four fingers on one of the user’s hands with cybernetic digits. When activated, these fingers each split in half, and then each of those then split again, resulting in sixteen independently operating digits controlled by an implanted sequencing processor. When the user touches these digits to a keypad, they tap through possible codes in a blur of motion. The system also broadcasts a short-range RFID signal that spoofs the device into overriding its own timeout functions that would ordinarily prevent rapid-fire code entries. Keypad sequencing fingers make an Opposed test, rolling its rating vs. the maglock’s rating; it doesn’t require the user to first open the

maglock’s protective housing. See Maglocks, p. 241, _SR6_ .

**Cyberware overdrive:** Gain +2 dice, one of which must be a wild die, on a test made to defeat a keypad maglock.

**Wireless bonus:** Gain a point of Edge when using an implanted maglock sequencer.
