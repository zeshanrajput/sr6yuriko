## Code Eco

A hacker with connections to the world of ecoterrorism, Code Eco was a good friend of Okoro’s and an even better friend after giving her the Kechibi Code. Code Eco found the code by chance (or at least, so he thinks). He snagged it as paydata from a run against an infobroker in London. It seemed like it had potential, and he gave it to Okoro as a small gift. He did not expect it to do what it did for her. Fate was not kind to Code Eco, and Lagos’ frequent random street violence found him before he could share the joy (and pains) of Okoro’s windfall. His body was pawned off to Tamanous, who split it between recovered cyberware and ghoul chow. His cyberware included a headware memory module that he used to store his most important data. Code Eco is dead, but the runners could get the tracker code for the RFID chip in his wares from one of the lawyers’ offices or from Okoro if she is the one hiring them or if they get access to her commlink.
