## Chassis

Select a chassis; this determines the base type of vehicle and gives the starting values for Body, Armor, Handling, Seating, and Size Class.

Size Class represents the sheer size and mass of a vehicle, and it also determines the baseline amount of cargo space available. Cargo capacity is measured in CFs (Cargo Factor); a single CF represents a half cubic meter of volume and 200 kg, and 1 CF is the approximate size of a typical human. The length and weight of each size class are approximate.
