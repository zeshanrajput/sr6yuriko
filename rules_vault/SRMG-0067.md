## **Can I roll my skill test for downtime activities?**

No. Any tests done as part of a downtime activity, such as a Matrix Search to find gear or to Initiate, can only be done using the “buying hits” method (SR6, pg 36). Additionally, anything that is permanent (such as Quickened spells) or things that last longer than one mission must also use the “buying hits” method. It is possible to roll dice to bind or register, but the set of services/tasks go away at the end of the _Mission_ .
