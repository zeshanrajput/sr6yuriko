## Shops, Facilities, and Automated Assistance Software

Automated assistance software (AAS) installed on toolkits, shops and facilities provides real-time analysis of a repair job, generating +1 Edge to all repair tests using those tools.

In addition, the AAS in a shop or facility allows runners to gain Edge beyond what they would normally be able to in a turn, as if the shop or facility were an independent character with their own Edge rating. Shops can store up to 4 Edge, and facilities can store up to 7. This Edge can be used by pit crews or drone assistants working in the shop or facility, and it can only be used on the project that generated it. Automated assistance software in a shop has Data Processing 4 and Firewall 5, while AAS in a facility has Data Processing 5 and Firewall 6.
