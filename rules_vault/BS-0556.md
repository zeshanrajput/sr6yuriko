## Awakened Critter Power Enhancement

An Awakened critter’s magical powers can be improved through this genetic augmentation. Only powers that use a critter’s Magic attribute as part of a dice pool may be enhanced, and the augmentation can only affect a single specified power— multiple applications of this augmentation can be used to improve multiple critter powers, however. Add the rating of this augmentation to the critter’s dice pool for tests that involve the specified power. Unless the critter receives special training to selectively use this augmentation, it will always apply the bonus when using the power.
