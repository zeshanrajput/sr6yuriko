## Electronics Systems Module

An electronics systems module is an external mount for electronic devices. It provides attachment points for any electronic devices, as well as rechargeable batteries with enough power to run those devices for up to twenty-four hours. Devices commonly attached to this module include commlinks, sensor arrays, jammers, and bug scanners, but nearly any electronic device is eligible.

**Game Information:** An electronic systems module can mount one device per point of its rating and takes up exoframe capacity equal to its rating. The maximum rating of any single device that can be attached is 6. If the critter is augmented with a critterjack, these devices can be integrated via DNI, allowing the critter to understand and make use of the information provided by the augmentations—otherwise, the information is only useful to users with access to the electronics systems. In order for a critter to be able to activate or use a device upon command, it must be trained to follow a special command.

**Wireless Bonus:** Electronic devices can be activated and controlled remotely via the Control Device Matrix action.

**Wireless Bonus:** The flash module may be activated and deactivated remotely using the Control Device Matrix action.
