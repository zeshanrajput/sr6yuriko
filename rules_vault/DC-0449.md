## Medullan Parallel Processors

This cyberware adds additional circuitry and nanoprocessors to the brain stem, and it must be used in conjunction with a control rig (p. 283, _SR6_ ). With this augmentation, the user is able to accept and react to a broader spectrum of simsense. This results in a deeper connection with a jumped-in vehicle than otherwise possible. Each level adds 1D6 initiative, which stacks with cold sim and hot sim VR user modes. This bonus only applies while the user is jumped-in.
