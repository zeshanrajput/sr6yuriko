## **Are all hosts in a nested host network part of the same network like a PAN?**

They aren’t. Each individual host is a network of devices, files, and agents that share access levels, but these aren’t shared with the sub-host that is attached to it. However, once you gain access to a host and the subhost you can take actions on both. Just as a fun reminder, your Overwatch is growing based on both those hosts’ access level. You can control the growth of OS by exiting subhosts, but you must retain access to the hierarchy.
