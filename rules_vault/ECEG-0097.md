## Flashlight, UV

This mod connects a UV fl ashlight to the underbarrel of any fi rearm. While it can function the same as any standard fl ashlight, its primary use is combating an HMHVV threat by triggering their allergy to sunlight. In combat, keeping the light trained on a moving target requires a Minor Action each turn. If the creature is in full cover, the effect doesn’t work.
