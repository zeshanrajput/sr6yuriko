## **How does Discreet Smuggler (6WC, p. 133) work when there is no palming test in** _**Sixth Edition**_ **?**

Replace the Game Effect with the following: When you take a Major Action to conceal an item, you increase the Concealability Threshold by 1. This is not cumulative. The size of the item can be no larger than a heavy pistol. This works against both Perception tests and scans using sensors.
