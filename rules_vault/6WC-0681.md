## Cover Shields Blasts

**Use this rule** to allow victims of a blast attack to claim some extra protection from cover.

Reduce blast DVs by two times the level of cover a character benefits from. This cover must be relative to the blast rather than relative to the attacker’s position. Additionally, if a character has no cover but has the Prone status (p. 53, _SR6_ ) reduce Close and Near blast DVs by 2. This DV reduction is an extension of this Hit the Dirt action (p. 41, _SR6_ ) and therefore does not stack with it.
