## Selective Dampening

Instead of enhancing a critter power, this applies a dice pool penalty equal to the rating of the genetic augmentation to a critter’s dice pool when it uses a specified power. This can only affect a single specified power, but multiple applications can be used to affect more than one power. The critter can receive special training to use its power without the dampening effect, which requires it to takean extra Minor Action before using its power. This is a popular option for security barghests, so they do not disable other security forces with their paralyzing howl when they are deployed.
