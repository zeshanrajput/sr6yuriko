## Game Information

Pixel tags emit light and are capable of producing nearly any color. The light’s intensity can be varied, but the brightest setting isn’t capable of inflicting any conditions. A single pixel tag is only capable of creating a single dot of light. though information and images can be conveyed through programmed pulses. Using multiple tags in concert

with a controller device allows for a wider range of visual possibilities. For a simple image, such as displaying text and simple icons, each square centimeter requires approximately ten pixel tags. For low resolution images (photo-realistic images are blurry or pixelated), each square centimeter requires approximately one hundred pixel tags. For realistic, high-resolution images, approximately 1,000-pixel tags per square centimeter are required.

Pixel tags are easily jammed, losing their connection to the Matrix in the presence of 2 or more points of noise.

To create an illusion using pixel tags, the appropriate amount of tags must be employed. Users are required to make a Con (Performance) + Charisma test, with the user’s hits setting the threshold for Perception tests to realize the illusion is fake. Simple two-dimensional illusions work best, such as editing a document, sign, or view screen. Three-dimensional illusions do not hold up to close scrutiny. Any Perception test made to discover the reality of a 3-D pixel tag illusion within a range of two meters or less automatically succeeds.

Note that pixel tags cannot move on their own. A 3-D pixel tag illusion requires an appropriate surface, and if that surface moves, so do the tags. They are capable of tracking their position relative to one another, however, which allows the illusion to be modified on-the-fly without the need for a complex and time-consuming recalibration.
