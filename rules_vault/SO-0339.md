## Smart Cosmetics

Smart cosmetics are makeup that links to a PAN, so that its status can be monitored, and colors can be adjusted after application. This system allows the user to know if your mascara is running or somehow compromised and allows for subtle adjustments or repairs to be made as desired. Useful for aesthetic versatility but it can also be used for a quick, subtle disguise.
