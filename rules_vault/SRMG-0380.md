## **What are the limitations to Threat Analysis (p. 31)?**

The bonus dice for this can never stack with another Threat Analysis. If a single decker uses this action multiple times in a single turn, it only uses the highest value. If multiple deckers use this action during a single round, it only uses the highest value. The recipient must also have a DNI or both image link and soundlink. This does not provide a bonus against magical attacks.
