## Hand and Foot Webbing

A simple implantation, added in pairs, adds webbing between the user’s toes or fingers, which are expanded by simple muscle contraction and withdrawn the same way (each is a Minor Action), allowing the user improved speed while swimming.

This provides a +1 dice pool bonus to all Swimming tests for each set installed but inflicts a –1 dice pool modifier to any fine manipulations using the webbed digits.
