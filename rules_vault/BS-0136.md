## Built-In Medkit

Just as cyberdecks or commlinks can be implanted into a skull, medkits can be built into a full or partial cyberlimb. The limb becomes equipped with useful surgical implements and a smart system to help with diagnoses and treatment. This system only involves the housing and connections for cybernetic control; a medkit (p. 281, _SR6_ ) still must be purchased separately for installation.

**Wireless bonus** : Same as a standard medkit. Additionally, if the user is in a VR Matrix user mode, they can employ a wireless built-in medkit on their own comatose body.
