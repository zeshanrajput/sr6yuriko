## Nano-Critter Bio-Drone Harness

A micronized version of the bio-drone implant allows a tiny critter to be controlled as if it were a drone. This does not provide a Pilot rating or the capacity to run programs, so the critter must be linked to an RCC, which overrides its will. The bio-drone is also incapable of being jumped into, but it can be directly controlled by the RCC. No autosofts are compatible with this augmentation, so the dice pool for all actions is either equal the RCC or the controller’s Piloting + Reaction. The level of control is relatively imprecise, so no Edge may be gained or spent on any tests for controlling a nano-critter bio-drone.
