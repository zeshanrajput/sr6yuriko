## Game Information

Provided that the user has a camera, optical device with an image link, or cybereyes, they can create a detailed scan of another person’s face with a Minor Action if that person is observable. This facial scan can then be used in conjunction with a nanopaste disguise. If someone attempts a Matrix Search to identify a person using the facial scan, gain a point of Edge. This Edge must be used on the test or is lost.
