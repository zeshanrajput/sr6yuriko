## True Fire

True Fire, or longqi (dragon breath), is unusually tangible. It has an appearance as a pale flame, giving off light and heat, but solid enough so that it can be picked up by tongs, provided that tool can resist the heat of a forge. It can only be quenched by True Water, though doing so is extremely dangerous. If you thought descending into a caldera on Earth to find longqi was difficult, imagine the molten cyclones that you’d have to descend into to find some remnants.
