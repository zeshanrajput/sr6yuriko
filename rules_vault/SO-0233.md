## Charm

(Illusion)

**==> picture [227 x 20] intentionally omitted \<==**

**----- Start of picture text -----**<br>
RANGE TYPE DURATION DV<br>Touch M S 5<br>**----- End of picture text -----**<br>

The Charm spell gives the target a silver tongue. Roll Sorcery + Magic and record the number of hits; those hits are used as a threshold when people try to see through the illusion, which requires a Willpower + Intuition test. While the Charm is in effect, the target gains +4 on all Con and Influence skill tests, and net hits on the Spellcasting test are added to the subject’s Social Rating (p. 50, _Firing Squad_ ) as long as the spell is sustained.
