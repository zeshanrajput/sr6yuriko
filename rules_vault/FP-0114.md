## **Ghostwalker’s Digital Hoard**

**After being opened to the Matrix by hackers, Ghostwalker cut the hardline connection to the Dragon’s Lair, making it an offine system. Eventually, this proved to be an unworkable solution, and SilveryK, now using the name Prophetess, set the Dragon’s Lair up as a nested host inside Anasazi Holdings. This host was built to manage Ghostwalker’s purchased properties and was already very secure (Rating 9, Grade 5, Scale 4). She constructed a system of rotating concealed gates inside the Anasazi Holdings host, with killing jars and trap doors connected to the Dragon’s Lair that duplicate the security setup of a data haven. The only people with the access codes and the location**

**of the gates are Ghostwalker’s most loyal technomancers. Anyone else who managed to fnd one of the concealed gates to a killing jar would most likely be dumpshocked while trying to get through.**
