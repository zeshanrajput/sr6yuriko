## Reality Filter

## Core Slots: 2

A reality filter is designed to translate all Matrix iconography to fit a programmed aesthetic. In addition to allowing you to make the Matrix look like the fantasy world of your choosing, you’re able to set up key iconography that allows you to more quickly process and respond to what you’re seeing. If you have an active reality filter and are using a cyberhack, you may use the Matrix Edge actions that normally require an implanted cyberjack. Other possible effects of this tech are unlocked through positive and negative qualities.
