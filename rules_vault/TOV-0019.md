## Conjuri ng Gris-Gris

Calling upon the lwa to assist practitioners of Voodoo is a core element of the religion. These charms prepare symbols that aid in the rituals to call spirits. The specifi c symbols used, and the type of spirit chosen only work on the specifi c type of lwa. Using such a grisgris that has the symbols and spirit of Ogou Batala will not help you when you beg Agwe for assistance. These symbols are often placed around the summoning space.

When activated, a conjuring gri-gris provides assistance on a skill test as a Teamwork test using Force x 2 as its dice pool. Conjuring gris-gris are specifi c to the type of lwa being called upon for help. Regardless of the success of the summoner’s test, all active grisgris expend one of their services to help with a Teamwork test as part of either Summoning or Banishing. There is no limit to the number of Conjuring gris-gris that can be used, but the summoner can only accept help up to their rank in the Conjuring skill—as with any other Teamwork Test.

For example: a street samurai has purchased three luck gris-gris (Force 2). They can use a Minor Anytime Action to activate all three gris-gris, but only a single one can be used to donate Edge on the next test.
