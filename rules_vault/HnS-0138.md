## Agent Box Hack

Agent software runs best on a cyberdeck, but a hacked commlink can be designed for use as a separate device made to host and run an agent program independently of its host cyberdeck. Hacking a commlink into an agent box requires an Electronics (Hardware) + Logic Extended test with a threshold equal to double the commlink’s device rating and an interval of one hour. Once transformed, the commlink becomes an agent box with a rating equal to the original commlink’s device rating. An agent box acts as an external program carrier for an agent with a rating no higher than the rating of the agent box. Detailed rules for agents, including specific uses for agent boxes, are in the **Digital Toolbox** chapter (p. 55).
