## General Rules for Conversion 

While there is a lot of common material shared by _SR5_ and _SR6_ , the core rulebook is just one book. With six years’ worth of _SR5_ material, there are far more abilities, tech, qualities, and magic than can be carried over into a single new core rulebook. To help with all those things we can’t exactly cover, we’re going to use the following Four Rules for Conversions for all of that additional material.
