## Program Name: The Kechibi Code

- **Type:** Special utility parameter (Illegal)

- **Price:** You cannot buy access to this program.

The code is powerful, invasive, and encrypted to levels not seen in most parts of the Matrix. Its complexity has evaded complete analysis by most, and because of the conflicts that arise over its existence, there are agencies that will hunt down those who have it, control it, or use it. They do not deal gently with those individuals. Reverse-engineering will thwart almost all attempts, because its design is high level and highly idiosyncratic. The code completely takes over any device or host that executes it; it is not possible for analysis programs to load and run concurrently.

The code has a built-in decay mechanism if it is copied or not used. This means copies made by anyone besides the designers or the code itself are

unusable, and unused copies of the code are unusable after one week. For information on gleaning information from decayed copies of the code, see **Decaying Kechibi Code** , below.
