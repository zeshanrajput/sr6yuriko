## **For the new sprite commands (p. 74-75), not all things can be treated as a teamwork test, how should treat those?**

Most of these commands work just fine with a teamwork test and the limits it imposes, but the Courier Sprite Task and the Fault Sprite Task don’t necessarily work as well. For Signal Boost, it works as it says, but only helps the technomancer and not everyone on the PAN. For Cybercombat Boost, the damage boost is half the sprite’s Resonance (rounded up).
