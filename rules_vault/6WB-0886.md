## Watermark

The sprite can tag an icon with an invisible marking that only Resonance-driven entities can see, kind of like a Matrix signature. This allows the sprite to secretly leave messages on Matrix objects. A sprite can overwrite an existing watermark with a new one. A watermark can be erased with the Erase Matrix Signature action by a Resonant being; otherwise it lasts as long as the icon does.
