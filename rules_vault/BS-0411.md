## Witch’s Moss

- **Vector:** Ingestion

- **Speed:** Immediate

- **Duration:** 1D6 hours

- **Power:** 8

- **Effect:** Paralyzing Touch (p. 227, _SR6_ )

Only a select few know the recipe for this magical compound, which resembles a green, porridge-like substance. The unappetizing paste grants the user the ability to paralyze with a touch. When the drug wears off, the user gains the Stilled status for an equal duration.
