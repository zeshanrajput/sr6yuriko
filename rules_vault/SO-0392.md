## Response Edge Actions

Most Edge actions augment a specific action that’s already been taken or attempted. Response Edge actions are different in that they’re triggered by someone else’s action, or by a specific circumstance. These all require an Anytime action (some require a Major Action, others a Minor Action), which must be performed immediately when the opportunity presents itself. A response Edge action can’t be used several rounds after the events that triggered it—it’s now or never. Some response Edge actions allow the user to salvage something, or even turn the tables, in reaction to an unfortunate turn of events, while others build upon or dovetail into someone else’s success.

Unlike most actions, response Edge actions don’t require or involve a dice roll. Simply spend the action and the Edge, and the effect happens. Most response Edge actions have a variable effect based on how much Edge is spent.
