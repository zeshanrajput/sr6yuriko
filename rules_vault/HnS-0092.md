## Core Slots

A cyberdeck has a number of available core slots equal to its Device Rating x 3. Each custom component takes up one or more core slots, but so do any connected devices. Note how many core slots are taken up by all of your cyberdeck’s components, because this reduces how many wireless (and wired) devices you can connect to your deck’s PAN.
