## drone watching

Some of the joggers in the area are actually anthroform drones. Spotting this requires an Electronics (Hardware) + Intuition (4) test using sensors or Perception (Visual) + Intuition (5) using sight. Use the stats for the Shiawase Bidrone Man-at-Arms (p. 117, Double Clutch). If Animus notices any attempts to hack the drones, they will immediately call NYPD, Inc.
