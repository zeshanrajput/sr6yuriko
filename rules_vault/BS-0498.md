## Dog’s Ears

The critter’s ears are enhanced to allow for much greater sensitivity, allowing them to perceive ultrasonic frequencies as well as enhancing normal hearing.

**Game Information:** The critter gains the Enhanced Senses (Hearing) critter power. This provides the critter with a point of Edge on Perception tests that use hearing, which must be used on the test or is lost. If the critter normally does not have an Edge attribute, the point of Edge is used to reroll a single failed die on the test.
