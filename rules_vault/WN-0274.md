## **Mods:** Stealthlink Upgrade 3

**Programs:** File Vault, Mannequin, Personal Assistant, P-ICE (NOPE!)

Should one of these commlinks be kept by a PC, it will need its ownership changed and the Stealthlink Upgrade reprogrammed. This is an Electronics (Hardware) + Logic (24, 1 hour) test. This also requires an Electronics toolkit and a coding kit (p. 56, Hack & Slash). If the number of hours in reprogramming exceeds 5, the commlink will self-destruct.

Tiding agents at this point are not out to kill.
