## Architect (The Builder)

Not as interested in the result of what they build, the Architect is invested in the act of creation. Once the well-designed and integrated code or infrastructure is complete, the architect loses interest and moves on to their next big project.

**Advantage:** Edge Boosts cost 1 less for Edit actions. **Disadvantage:** Cannot gain or spend Edge for Snoop actions. Favor is lost if the technomancer has not created software for a while, and it is lost faster if the last software created was not a challenging project. Favor can be regained with a large coding project.
