## Stick to the Plan

## Posted by: High Road

Okay, let’s talk about the importance of planning. Now, I’m not talking about planning every step of a run. Rather, I want to focus specifically on the social components of planning a run and how they’re not just critical to team cohesion, but also to the execution of that plan.

And because everyone loves heists (I know I do), I’m going to focus on that type of run to conceptualize and illustrate my key points. So, while I’ll be talking about planning a heist, the data I’m about to drop can be used on any type of run that requires gathering data and information— which is all of them.
