## Build Costs

**Priority Build:** Monad joins the column for Magic, Resonance, and Mundane. Starting NV Attribute is based on the same as the other special attributes and increased the same way. Monad PCs start with 1 Monad Ability per NV. Future Monad Abilities are bought with 10 Karma for each level.

**Point Build:** Monad is 10 CP, just like Awakened or Emerged.

**Life Path:** Each place there is an Awakened or Emerged option, a similar option can be gained for a Monad.
