## Optional Components

## Internal Program Slot

## Core Slots: 1 each

A program slot is a data storage device and processor chip that augments a cyberdeck with the ability to run a cyberprogram. A custom cyberdeck starts with a number of program slots equal to its device rating, but you can add more at the expense of core slots. Each additional internal program slot takes up one CPU port.
