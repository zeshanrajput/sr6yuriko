## Vamping

## Posted by: Red

So I can see why Fianchetto calls Snares passive. This one is more like one of my favorite scifi novels called _Follow Best_ , where you’re hitting the streets looking like a big, fat target. You’re actively soliciting trouble, and you’re set up to handle it when it comes, to your profit. Obviously someone with an inherent capacity to regenerate and the desire to clean up the streets is going to have an easier time, so I don’t expect anyone to go it alone like I might, but integrating a team is pretty simple.
