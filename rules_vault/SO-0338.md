## Game Information

Wearing poser props allows the user to disguise themself as the specified metatype. At a glance or cursory inspection, most people tend to assume that the user is the metatype they’re portraying. If the user’s appearance is scrutinized, the user gains a point of Edge when making Con or Stealth skill tests to pass as the specified metatype. This Edge must be spent on the test or it is lost.
